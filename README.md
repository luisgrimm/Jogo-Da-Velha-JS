# 🎮 Jogo da Velha

Desenvolvi um projeto simples de **Jogo da Velha (Tic-Tac-Toe)** utilizando **HTML, CSS e JavaScript**, com foco em praticar manipulação de DOM e lógica básica de programação.

## 📌 Funcionalidades

- Tabuleiro 3x3 interativo  
- Alternância automática entre jogadores (**X** e **O**)  
- Cada célula pode ser clicada apenas uma vez  
- Botão para reiniciar o jogo  

## 🛠️ Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript  

## ▶️ Como executar

1. Baixe ou clone o projeto  
2. Abra o arquivo `index.html` no navegador  

## 📂 Estrutura do projeto

- `index.html` → estrutura da página  
- `style.css` → estilos do jogo  
- `script.js` → lógica do jogo  

---

💡 Projeto desenvolvido para reforçar conceitos fundamentais de JavaScript e interação com o usuário.
